Você sabe o que é a programação web ?

É a área de desenvolvimento de software que foca na criação de programas para a internet como: sites de vendas, notícias, aplicativos de namoro, banco, entre variadas opções. Pode parecer algo extremamente complexo, mas nosso objetivo é ajudar futuros devs que desejam aprender ainda mais sobre o mundo da programação. Lembrando que o mercado para o desenvolvimento web continua a crescer todos os anos, com remunerações altas, flexibilidade em horários e oportunidade de ascensão.

É difícil aprender as linguagens da web ?

A resposta é não, o HTML e CSS são "linguagens" com características únicas que não vão causar confusão caso você saiba algo como python, java, php, enfim. Mas vale ressaltar que para Javascript é necessário que você tenha estudado os conceitos fundamentais da programação como algoritmos, é base de toda e qualquer linguagem, então se futuramente você desejar acessar outros caminhos nesse vasto mundo, é fundamental saber conceitos basilares. Muitas vezes priorizar o que deve ser estudado pode ser complexo, então mantenha a calma e venha com nosso passo a passo.

Divisões da Web ?

Sim, a programação para web é separada em dois segmentos, front-end (frente) e o back-end (fundo). Vamos com calma para saber um pouco de cada. O front-end é área que vai ser responsável por criar todos os componentes visuais do programa, então tudo que você consegue ver e interagir como botões, cor, barra de pesquisa, cadastro, entre outros, é de responsabilidade do programador front-end. E o programador back-end ? Ele é incumbido de construir a parte mais lógica das aplicações, vão criar e gerenciar o banco de dados, colocar a aplicação na rede, cuidar da manutenção do software no servidor e proteger os dados na rede. O que preciso saber para ser um dev front-end ou back-end ?
